# Streaming App

Bienvenue dans l'application de streaming Steaming de Lemming !

## Description

Streaming App est une application de streaming vidéo où les utilisateurs peuvent uploader et regarder leurs vidéos préférées.

## Fonctionnalités

- Uploader des vidéos
- Regarder des vidéos en streaming
- Liste de vidéos disponibles


## Installation

1. Installez les dépendances : `npm install`
2. Démarrez le serveur : `npm start`

## Configuration

- Le port par défaut est configuré pour être 3000.
- Assurez-vous d'avoir Node.js et npm installés.
